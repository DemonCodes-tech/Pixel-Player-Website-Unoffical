# Pixel Player: The Website (Unofficial)

**Pixel Player: The Website (Unofficial)** is a lightweight, web-based offline music player built on the refined architecture of the open-source Pixel Player Android app. Designed for music lovers who value autonomy, it bridges the gap between local playback and web utility, allowing you to curate your perfect library without relying on constant internet connectivity.

---

### 🚀 Key Features

* **Offline First, Online Flexible:** Upload your personal audio files directly into the web interface and enjoy seamless, interruption-free playback entirely offline.
* **YouTube Integration:** Instantly expand your library by downloading and converting audio straight from a YouTube link directly inside the app.
* **Material Design DNA:** Inherits the clean, intuitive, and highly customizable UI aesthetics of the classic Pixel Player.

---

### 🗺️ Future Roadmap: Moving to Desktop

While currently operating as a powerful web application, the long-term vision for **Pixel Player: The Website (Unofficial)** is a complete transition into a **standalone desktop application** (via frameworks like Electron or Tauri). This shift will unlock:

* Deeper system integration and global media hotkeys.
* Advanced local file-system syncing.
* Enhanced performance and dedicated background playback.
